# ClassLite - Initial Feature Request & MVP Requirements

**Source:** User Chat Input
**Date:** 2026-01-16

## Project Overview
ClassLite is an LMS that is made to target small to medium IELTS English centers in the Vietnamese market. We plan to expand to other types of education like Math, Physics later on.

## Proposed Features (MVP)

### Authentication
- Oauth authentication (Google, Facebook)
- Multiple roles authorization and permissions (Center/Admin/Teacher/Student)

### Class & Student management
- Add users to centers by email invitation
- Create classes or courses, add users to classes, schedule classes
- End class, delete users

### Class schedule
- Google Calendar like UI view for classes schedule

### Lesson notes/recording
- Teacher can add notes and files for each lesson, student can review them

### Attendance tracking
- (Implicit: tracking student attendance per class/lesson)

### Knowledge Hub
- Long living documents the center can build for its students. Like a library for each center

### Exercise builder
- Dynamic exercise builder to build IELTS exercises
- Import from pdf, word document, use AI to process into exercises

### Exercise Assignment
- Assign exercises for students/classes. Assign homework for lessons

### Exercise Submission
- Student completes exercises for teachers to review

### Exercise Grading/Feedback
- Auto grading for Reading and Listening
- AI assisted grading for Writing and Speaking

### Notifications
- Class schedule reminders
- Assignment reminders
